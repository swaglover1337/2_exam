# 2_exam
Проект «Разработка информационной системы молочного комбината»
Система предназначена для работы с пользователями системы (управление, добавление, изменение, блокировка).
Реализованный функционал:
- Авторизация в системе с помощью логина и пароля
- Возможность администратора системы управлять пользователями
- Страница пользователя система
Реализовано с помощью SQL, Visual Studio и дополнительных графических программных средств
Приложение запускается файлом .exe
